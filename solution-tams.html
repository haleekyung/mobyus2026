TAMS

통합 관제 시스템

TAMS는 다수·이기종 로봇과 설비, 작업 흐름까지 통합 조율해 교착을 막고 물류 현장의 전체 효율을 최적화하는 '통합 관제 시스템' 입니다.

왜 TAMS가 필요한가

여러 로봇/기기 혼재 해결
기기간 교착 제어
로봇/설비간 충돌 및 대기
시스템별 제어 통합

TAMS의 핵심기능 5가지

1. 300대 이상 동시제어
  • - 300대 동시접속 (1초내 대기 기준)
  • - 메모리 캐시(Message Queue) 기반 정보 관리
  • - Microservices 기반 서비스 관리
2. 이기종 통합 관제
  • - 청소로봇, 타사 AGV, 유인지게차 등 이기종 충돌 방지하고, 작업 조율
  • - WNS 모듈로 유인장비의 위치를 실시간 추적, 안전사고 예방
3. AMR 이송 최적화
  • - AI 기반 물동 사전 물동 예측
  • - 군집 이동 최적화 (대열줄맞춤, 작업 할당 매칭)
  • - 경로 최적화 (최적 경로탐색 및 혼잡구간 우회)
4. Realtime 현장실행 및 대응
  • - 중앙 3D 관제 포털 제공
  • - Mobile 기반 UI 및 키오스크 UI 환경 제공
  • - 플랫폼 기반 (UI개발 플랫폼, CAD 플랫폼)
5. 사후 관리 및 분석
  • - 관리자 타겟의 분석툴
  • - KPI 관리
